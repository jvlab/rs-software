# Demos

The library contains several demos that show its functionality. They are located in folder `src/demos`.


- [`Contents`](mfiles/demos/Contents.md) - Representational space software demos
- [`rs_align_disp_coordsets_demo`](mfiles/demos/rs_align_disp_coordsets_demo.md) - demonstration workflow for display of several datasets and their consensus
- [`rs_align_disp_enh_coordsets_demo`](mfiles/demos/rs_align_disp_enh_coordsets_demo.md) - demonstrate display of datasets with offsets, rays, and other enhanced plotting options
- [`rs_disp_coordsets_demo`](mfiles/demos/rs_disp_coordsets_demo.md) - simple display of coordinate sets
- [`rs_disp_coordsets_demo_cars`](mfiles/demos/rs_disp_coordsets_demo_cars.md) - 
- [`rs_disp_coordsets_demo_opposites`](mfiles/demos/rs_disp_coordsets_demo_opposites.md) - rs_disp_coordsets_demo_opposites
- [`rs_knit_coordsets_demo`](mfiles/demos/rs_knit_coordsets_demo.md) - combine coordinate sets with partially overlapping stimuli
- [`rs_read_coorddata_demo_cars`](mfiles/demos/rs_read_coorddata_demo_cars.md) - rs_read_coorddata_demo_cars
- [`rs_read_coorddata_demo_opposites`](mfiles/demos/rs_read_coorddata_demo_opposites.md) - rs_read_coorddata_demo_opposites
- [`rs_toygeom_demo`](mfiles/demos/rs_toygeom_demo.md) - geometric modeling with toy simulated datasets; used by rs_toygeom_scenario*
- [`rs_toygeom_disp`](mfiles/demos/rs_toygeom_disp.md) - display the results of rs_toygeom_demo* with `rs_disp_geofit`
- [`rs_toygeom_scenarioA`](mfiles/demos/rs_toygeom_scenarioA.md) - geometric transformations, without statistics: procrustes, affine, projective, piecewise affine
- [`rs_toygeom_scenarioB`](mfiles/demos/rs_toygeom_scenarioB.md) - geometric transformations, with statistical analysis of nested models: procrustes, affine, projective
- [`rs_toygeom_scenarioC`](mfiles/demos/rs_toygeom_scenarioC.md) - geometric transformations, affine model, with statistical analysis of nesting by output dimension
- [`rs_toygeom_scenarioD`](mfiles/demos/rs_toygeom_scenarioD.md) - geometric transformations, affine model, with statistical analysis of nesting by input dimension
- [`rs_xform_coordsets_demo`](mfiles/demos/rs_xform_coordsets_demo.md) - demonstrate specification and application of simple linear transformations
